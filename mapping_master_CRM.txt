Notes Updation

To programmatically add a note to a contact via the Freshsales API, you would use the Notes API endpoint.
Programmatically Adding a Note to a Contact via Freshsales API:
You can create a new note and associate it with a contact using a POST request to the /api/notes endpoint.

Authentication: You will need your Freshsales API key for authentication, typically included in the Authorization header.
Request URL:

Code

https://.freshsales.io/api/notes

Replace <your_domain> with your Freshsales account's subdomain. Request Body (JSON).
The request body should contain the note details and the ID of the contact it should be associated with.
Code

{
"note": {
"description": "This is a new note for the contact.",
"targetable_id": <contact_id>,
"targetable_type": "Contact"
}
}

description: The content of your note.
targetable_id: The unique ID of the contact you want to add the note to.
targetable_type: This must be "Contact" to associate the note with a contact.

Example using cURL:
Code

curl -X POST 'https://.freshsales.io/api/notes' \
-H 'Authorization: Token token=<your_api_key>' \
-H 'Content-Type: application/json' \
-d '{
"note": {
"description": "Discussed product features and next steps.",
"targetable_id": 402082323753,
"targetable_type": "Contact"
}
}‘

Replace <your_domain> and <your_api_key> with your actual Freshsales domain and API key, and <contact_id> with the ID of the specific contact.
